# CODEX briefs

This folder stores design reviews and consultations from Codex / ChatGPT.

## File naming

Use `YYYY-MM-DD-topic.md` (example: `2026-06-01-cleanup-sprint.md`).

## Required frontmatter

Each brief starts with:

```yaml
date: YYYY-MM-DD
topic: short-topic
decision-status: open | decided | superseded
```
